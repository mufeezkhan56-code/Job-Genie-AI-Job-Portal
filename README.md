# Job-Genie-AI-Job-Portal
AI-powered job portal built using Flask, Python and Machine Learning
Features:
- Resume upload & parsing
- Resume scoring
- Job matching using NLP
- Recruiter dashboard
- Job seeker dashboard
- Shortlisting system

Tech Stack:
- Python
- Flask
- SQLite
- Bootstrap
- NLP
- Scikit-learn

This project was built as my final year project.
